# Juan The Horse - $JUAN

A React-based website for the Juan The Horse memecoin.

## Getting Started

### Install Dependencies

```bash
npm install
```

### Development

Run the development server:

```bash
npm run dev
```

### Build

Build for production:

```bash
npm run build
```

### Preview

Preview the production build:

```bash
npm run preview
```

## Tech Stack

- React 18
- Vite
- React Router DOM
